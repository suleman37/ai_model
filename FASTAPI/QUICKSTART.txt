===== EAR SEGMENTATION FASTAPI - QUICK START =====

� WHAT YOUR MODEL DOES:
✓ Detects ear in your image using YOLOv8 instance segmentation
✓ Creates a binary mask around the ear
✓ Extracts and measures ear dimensions (height & width)
✓ Converts pixel measurements to centimeters automatically
✓ Returns 3 visual outputs showing the segmentation result
✓ Fast inference (~1-2 seconds per image on CPU)

�📁 FILES:
✓ app.py          - Single file with ALL processing
✓ best.pt         - Your trained model weights
✓ requirements.txt - Dependencies

===== SETUP =====

1. Install dependencies:
   pip install fastapi uvicorn ultralytics opencv-python pillow numpy python-multipart

2. Run the server:
   python app.py

   You should see:
   ✓ Model loaded successfully from best.pt
   INFO:     Uvicorn running on http://0.0.0.0:8000

===== API ENDPOINTS =====

📊 GET http://localhost:8000/
   - Check API status

💚 GET http://localhost:8000/health
   - Health check & model status

🎯 POST http://localhost:8000/predict
   - Upload single ear image
   - Returns: Height & Width + 3 visualizations

👂👂 POST http://localhost:8000/predict-dual
   - Upload LEFT and RIGHT ear images
   - Returns: Results for BOTH ears + comparison + 6 visualizations
   - NEW FEATURE!

===== USAGE =====

Via Swagger UI:
   - Open: http://localhost:8000/docs
   - Click "Try it out" on any endpoint
   - Upload your image file

Via cURL:
   curl -X POST "http://localhost:8000/predict-simple" \
     -F "file=@your_image.jpg"

Via Python:
   import requests
   with open("image.jpg", "rb") as f:
       response = requests.post("http://localhost:8000/predict-simple", files={"file": f})
       result = response.json()
       print(f"Height: {result['data']['height']['pixels']} px ({result['data']['height']['cm']} cm)")
       print(f"Width: {result['data']['width']['pixels']} px ({result['data']['width']['cm']} cm)")

===== 3 OUTPUT VISUALIZATIONS =====

1. segmentation_result
   - YOLO detection overlay on original image

2. segmented_ear
   - Extracted ear on black background

3. mask_visualization
   - Binary mask visualization

===== MAIN PARAMETERS =====

✅ height.pixels (vertical measurement in pixels)
✅ height.cm (vertical measurement in centimeters)
✅ width.pixels (horizontal measurement in pixels)
✅ width.cm (horizontal measurement in centimeters)

Plus 3 segmentation visualizations included in response

===== TROUBLESHOOTING =====

❌ Model not found:
   - Make sure best.pt is in the same folder as app.py

❌ "No ear detected":
   - Check image quality
   - Try lowering confidence: ?confidence=0.2

❌ Port 8000 already in use:
   - Edit the last line of app.py
   - Change: uvicorn.run(app, host="0.0.0.0", port=8080)
